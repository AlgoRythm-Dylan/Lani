Lani-CSS Version

The Gist

All Lani CSS classes are prefixed with l-. This is to help with compatibility with other CSS frameworks (like Bootstrap) and to simply not pollute the CSS global namespace.

Lani uses CSS3 variables so that you have dynamic access to all the values Lani uses. This also enables some pretty easy dynamic theming. CSS3 variables are prefixed with --lani

Many classes come with three different intensity values, each with their own postfix on the class name: strong, subtle, and normal, which has no postfix.

  • l-[class]-strong
  • l-[class]
  • l-[class]-subtle

Where [class] is one of: text, text-accent, rounded, shadow, grow
